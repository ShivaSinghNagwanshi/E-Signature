## Apple Mail Preview Mockup UI
The Apple Mail layout and preview UI implemented in this project are based on the [Apple Mail mockup design](https://www.figma.com/community/file/1204925937933845366) posted on Figma by **[Austin Condiff](https://www.figma.com/files/team/1652565454207649012/resources/community/@austincondiff)**. 

This original Figma design is licensed under the [Creative Commons Attribution 4.0 International License (CC BY 4.0)](https://creativecommons.org/licenses/by/4.0/). 

**Modifications Made:**
- Adapted the static Figma vector layout into a fully responsive HTML/CSS (Tailwind CSS) web interface.
- Modified the design to include a dark-mode theme.
- Added dynamic Javascript skeleton loaders and dynamic Jinja2 template bindings for use as an email signature preview tool.
- Omitted specific sidebar elements in favor of a responsive 3-pane layout toggle system.
